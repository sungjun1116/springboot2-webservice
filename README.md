# springboot2-webservice
스프링 부트와 AWS로 혼자 구현하는 웹 서비스

## 시스템 구성
- JDK 1.8
- Spring Boot 2.4.1
- Spring Security
- Spring Rest Docs
- Spring Data JPA
- mustache

## DB
local: h2

prod: AWS RDS
